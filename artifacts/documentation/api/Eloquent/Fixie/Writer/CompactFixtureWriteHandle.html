Eloquent\Fixie\Writer\CompactFixtureWriteHandle

class CompactFixtureWriteHandle extends AbstractWriteHandle

A writable data handle that writes rows in the 'compact' style, using the minimal amount of whitespace.

This variant is excellent for any data size, but is not as good for human readability as other options. If human readability is not an issue, use this variant.
